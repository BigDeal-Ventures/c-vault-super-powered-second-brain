# Agent Collaboration

Use these rules when an AI agent works in the vault.

## Working rules

- Read the relevant notes before proposing changes.
- Preserve user-written content unless asked to rewrite it.
- Do not move or delete notes without explicit approval.
- Keep edits scoped to the requested task.
- Clearly separate facts from assumptions.
- Leave a handoff note when work is incomplete.
- After changing files, report exactly what changed and how it was verified.

## Parallel work

Assume another person or agent may be editing nearby files. Before changing existing material, inspect the current file state. Do not revert work you did not make.

## Cross-tool compatibility

Write instructions in plain Markdown. Avoid syntax that only one assistant or editor understands unless it is isolated behind a generated adapter.
