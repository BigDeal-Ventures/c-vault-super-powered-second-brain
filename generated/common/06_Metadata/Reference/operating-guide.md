# Operating Guide

This pack provides a clean, reusable second-brain operating layer.

## Principles

- Capture quickly, organize deliberately.
- Keep active work easy to resume.
- Prefer durable notes over chat transcripts.
- Separate projects, areas, resources, archives, metadata, and client work.
- Make every handoff clear enough for a future agent or collaborator.

## Daily rhythm

1. Capture raw thoughts in `00_Inbox`.
2. Review urgent tasks and active project notes.
3. Write or update the daily note.
4. Move processed notes to their long-term home.
5. End the day with progress, blockers, and carry-forward items.

## Weekly rhythm

1. Review daily notes and active projects.
2. Summarize progress and decisions.
3. Close or archive stale loops.
4. Promote useful research into resources.
5. Choose the next week's primary outcomes.
