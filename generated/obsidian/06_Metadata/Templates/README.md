# Templates

[Vault Home](../../README.md) | [Metadata](../README.md)

Use this folder for reusable note templates.

Templates should be plain Markdown, portable across tools, and free of personal defaults. Use placeholders such as `{{date}}`, `{{project_name}}`, and `{{client_name}}`.

## Templates

- `daily-note.md`
- `project.md`
- `research-note.md`
- `client-readme.md`
- `decision-log.md`
