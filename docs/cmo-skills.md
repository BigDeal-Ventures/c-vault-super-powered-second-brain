# CMO Skills

The `cmo-skills` pack bundles CMO Skills / Compounding Marketing v1.6.0 into C-Vault. It is designed for B2B SaaS marketing strategy, research, positioning, content, copy, SEO, CRO, lifecycle marketing, launch planning, and revenue operations.

## Workflow Commands

Use `cm-*` commands when you want a guided multi-step process:

- `/cm-research`
- `/cm-position`
- `/cm-copy`
- `/cm-launch`
- `/cm-audit`
- `/cm-sprint`
- `/cm-daily`
- `/cm-weekly`
- `/cm-standup`
- `/cm-eod`
- `/cm-email`
- `/cm-social`
- `/cm-retro`
- `/cm-compound`
- `/cm-setup`
- `/cm-uninstall`

## Specialist Skills

Use specialist skills when you need a focused marketing deliverable. Examples include `content-strategy`, `positioning`, `copywriting`, `copy-editing`, `icp-research`, `competitive-analysis`, `gtm-strategy`, `seo-audit`, `ai-seo`, `page-cro`, `paid-ads`, `pricing-strategy`, `email-sequence`, `press-pr`, `sales-enablement`, and `revops`.

## MCP and Integrations

The pack includes public setup notes for MCP-backed research tools and marketing integrations. Generated installed docs place these under:

- `06_Metadata/Reference/CMO Skills/MCP`
- `06_Metadata/Reference/CMO Skills/Integrations`

These files contain placeholder credentials and environment variables such as `${EXA_API_KEY}`, `${PERPLEXITY_API_KEY}`, `lin_api_xxxx`, and sample token strings. Real secrets belong in local tool configuration, not committed docs.
