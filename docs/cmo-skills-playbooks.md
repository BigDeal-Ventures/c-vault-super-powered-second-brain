# CMO Skills Playbooks

Use these playbooks when the user describes a marketing goal rather than naming a command or skill.

## Launch

1. Run `/cm-research` if market, ICP, or category is unclear.
2. Run `/cm-position` to sharpen narrative and differentiation.
3. Use `/cm-launch` for timeline, channels, assets, owners, and launch-day execution.
4. Use `/cm-email`, `/cm-social`, `press-pr`, and `product-hunt-launch` for channel execution.
5. Run `/cm-retro` and `/cm-compound` after launch.

## Positioning and Messaging

1. Use `customer-research`, `icp-research`, and `competitive-analysis`.
2. Run `/cm-position` for the full workshop or `positioning` for a focused canvas.
3. Use `messaging-framework` and `value-proposition`.
4. Apply the messaging with `/cm-copy`, `copywriting`, or `copy-editing`.

## Content Engine

1. Use `content-strategy` for themes, journey mapping, formats, and calendar.
2. Use `seo-audit`, `ai-seo`, `site-architecture`, and `programmatic-seo`.
3. Use `social-content`, `social-media-strategy`, `newsletter-growth`, and `lead-magnets`.
4. Use `content-performance-scoring` and `/cm-weekly` to review performance.

## CRO and Funnel Improvement

1. Run `/cm-audit` to identify constraints.
2. Use `page-cro`, `form-cro`, `signup-flow-cro`, `onboarding-cro`, or `paywall-upgrade-cro`.
3. Use `ab-test-setup` for hypotheses and decision criteria.
4. Use `analytics-tracking` and `attribution-modeling` to measure outcomes.

## Lifecycle and Revenue

1. Use `email-sequence`, `marketing-automation`, and `email-deliverability`.
2. Use `churn-prevention`, `referral-program`, and `community-strategy`.
3. Use `sales-enablement`, `revops`, and `pricing-strategy`.
4. Use `/cm-weekly` and `/cm-retro` to compound learnings.
