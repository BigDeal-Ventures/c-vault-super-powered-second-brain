# CMO Skills Decision Guide

[Docs Home](../README.md)

Use this guide to choose the right CMO workflow or specialist skill.

| If the user asks... | Use | Why |
|---|---|---|
| Understand market, ICP, or competitors | `/cm-research`, `icp-research`, `competitive-analysis` | Research gathers evidence before strategy. |
| Position the product | `/cm-position`, `positioning` | Full workshop vs focused positioning canvas. |
| Build messaging or value props | `messaging-framework`, `value-proposition` | Converts strategy into reusable claims. |
| Plan content | `content-strategy` | Maps topics, formats, funnel stage, and distribution. |
| Write copy | `/cm-copy`, `copywriting`, `copy-editing` | End-to-end workflow vs focused drafting or editing. |
| Improve SEO or AI search | `seo-audit`, `ai-seo`, `site-architecture`, `schema-markup` | Search, answer engines, structure, and machine readability. |
| Improve conversion | `/cm-audit`, `page-cro`, `form-cro`, `signup-flow-cro`, `onboarding-cro` | Broad audit first, then surface-specific CRO. |
| Plan launch | `/cm-launch`, `launch-strategy`, `press-pr`, `product-hunt-launch` | Narrative, timeline, channels, and promotion. |
| Build lifecycle campaigns | `/cm-email`, `email-sequence`, `marketing-automation`, `email-deliverability` | Campaign structure, automation, and inbox placement. |
| Run marketing cadence | `/cm-sprint`, `/cm-daily`, `/cm-weekly`, `/cm-standup`, `/cm-eod` | Marketing operating rhythm. |
| Capture learnings | `/cm-compound` | Turns work into reusable marketing knowledge. |

If two routes fit, ask one short clarification before choosing.
