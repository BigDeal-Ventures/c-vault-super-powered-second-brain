# C-Vault Examples

[Docs Home](../README.md)

## Core

```text
/daily-review
/weekly-synthesis for this week
/inbox-processor for this note
/thinking-partner help me reason through this product idea
/de-ai-ify make this draft sound less AI
```

Natural asks:

```text
Synthesize this week's notes.
Clean this inbox note and suggest where it belongs.
Help me think through this positioning decision.
Make this writing sound more grounded.
```

## CMO Skills

```text
/cm-research for this new SaaS category
/cm-position for AuthorityMax
/cm-copy for this landing page
/cm-launch for a new feature release
```

Natural asks:

```text
Plan content for this product.
Research ICPs and competitors for this market.
Improve this homepage copy for conversion.
Build a launch plan for this feature.
```

## Routing

Direct slash commands are the most deterministic. Natural asks are routed through `using-c-vault`; if two routes are plausible, the agent should ask one short clarification.
