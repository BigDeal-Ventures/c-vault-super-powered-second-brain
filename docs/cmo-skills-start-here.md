# CMO Skills Start Here

[Docs Home](../README.md)

Use this when starting marketing work in a new project or when you are unsure which CMO Skills workflow to run first.

## Recommended Sequence

1. Run `cm-context` or `/cm-setup` to establish product, audience, positioning, competitors, and brand voice.
2. Run `/cm-research` or `icp-research` to understand customers, competitors, switching triggers, and buying criteria.
3. Run `/cm-position` or `positioning` to define alternatives, differentiation, category, and best-fit customers.
4. Use `messaging-framework` and `value-proposition` to turn strategy into reusable messages.
5. Use `content-strategy`, `copywriting`, `seo-audit`, `ai-seo`, `page-cro`, or lifecycle skills depending on the growth bottleneck.
6. Run `/cm-compound` after major work to capture reusable learnings.

## Fast Paths

| Situation | Start With | Then Use |
|---|---|---|
| New product or vague strategy | `/cm-research` | `/cm-position`, `messaging-framework`, `content-strategy` |
| Landing page or homepage copy | `/cm-copy` | `copywriting`, `copy-editing`, `page-cro` |
| Content engine | `content-strategy` | `ai-seo`, `seo-audit`, `programmatic-seo`, `social-content` |
| Launch | `/cm-launch` | `launch-strategy`, `press-pr`, `product-hunt-launch`, `/cm-social` |
| Funnel improvement | `/cm-audit` | `page-cro`, `signup-flow-cro`, `onboarding-cro`, `pricing-strategy` |
| Marketing rhythm | `/cm-sprint` | `/cm-daily`, `/cm-weekly`, `/cm-retro`, `/cm-compound` |
